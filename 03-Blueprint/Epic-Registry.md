# Epic Registry

Version: 0.1
Estado: Aprobado

## Objetivo
Definir las unidades principales de desarrollo funcional para Tarevo V1.

## Convencion

Cada Epic debe tener:

- Objetivo.
- Alcance.
- Pantallas relacionadas.
- Reglas de negocio.
- Permisos.
- Eventos.
- API.
- Entidades de datos.
- Definition of Done.
